# TBeeper Library
Very simple little library that allows you to have an object oriented buzzer. You of course can have a vector or array of these if you have more than 1 (maybe I could add in some sort of stereo feature in the future?).

## Examples
You'll find a 'demo.cpp' example in the 'examples' folder. This shows you how to setup a singular buzzer and interact with it. If you upload the example to an Arduino with a buzzer hooked up to the builtin led pin you'll get to hear a few different tones and chimes.

## Dependencies
- [TSoftwareVersion](https://github.com/tjcooper97/TSoftwareVersion): All my libraries use this so that projects I make can report what library versions they're using
